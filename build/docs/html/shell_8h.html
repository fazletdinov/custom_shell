Подробное описание

Основной заголовочный файл для пользовательской оболочки bash.

Автор
Custom Shell Team
Версия
1.0.0
Дата
2024

Этот файл содержит основные определения и структуры для пользовательской оболочки. Оболочка поддерживает базовые команды, переменные окружения и выполнение внешних программ.
